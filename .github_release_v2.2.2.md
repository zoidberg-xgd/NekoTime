# NekoTime v2.2.2

## 🐛 修复

### 用户体验改进
- **透明度保持**：修复显示窗口后透明度丢失问题
  - 从托盘菜单显示窗口时，现在会恢复用户配置的透明度
  - 不再固定为 100% 不透明，保持用户的个性化设置
  
- **锁定位置保护**：锁定位置后禁用双击隐藏
  - 当窗口位置被锁定时，双击不再触发隐藏
  - 避免锁定状态下的误操作

这是一个改进用户体验的补丁版本，解决了实际使用中发现的两个常见问题。

## 📦 下载

选择适合您操作系统的版本：

- **macOS**: `NekoTime-macOS-v2.2.2.dmg` 或 `NekoTime-macOS-v2.2.2.zip`
- **Windows**: `NekoTime-Windows-x64-v2.2.2.zip`
- **Linux**: `NekoTime-Linux-x64-v2.2.2.tar.gz`

## 🔧 安装说明

### macOS
1. 下载 `.dmg` 文件
2. 双击打开，将 NekoTime 拖入 Applications 文件夹
3. 首次运行可能需要在系统偏好设置中允许

### Windows
1. 下载并解压 `.zip` 文件
2. 运行 `NekoTime.exe`
3. Windows 可能提示 SmartScreen，点击"更多信息"后选择"仍要运行"

### Linux
```bash
# 解压
tar -xzf NekoTime-Linux-x64-v2.2.2.tar.gz
cd NekoTime-Linux-x64-v2.2.2

# 安装依赖（首次运行）
sudo ./scripts/install_linux_deps.sh

# 启动
./run_linux.sh
```
